# تطبيق مراجعة مفردات المحاسب القانوني

تطبيق Flask خفيف لعرض ومراجعة المحتوى العلمي من ملفات داخل المشروع، بدون قاعدة بيانات وبدون تسجيل دخول وبدون CRUD.

## التشغيل

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

ثم افتح:

```text
http://127.0.0.1:5000
```

## بنية المشروع

- `app.py`: نقطة التشغيل.
- `app/routes.py`: الراوتات الأساسية.
- `app/services/content_service.py`: اكتشاف المحتوى وقراءة الملفات.
- `templates/`: القوالب.
- `static/style.css`: تنسيق الواجهة.
- `content/`: المفردات الرئيسية والفرعية وملفات المحتوى.

## إضافة مفردة رئيسية جديدة

1. أنشئ مجلدًا جديدًا داخل `content/`.
2. أضف ملف `meta.json` للمفردة الرئيسية.
3. أنشئ داخله مجلدات المفردات الفرعية.

## إضافة مفردة فرعية جديدة

1. أنشئ مجلدًا فرعيًا داخل المفردة الرئيسية.
2. أضف ملف `meta.json` للمفردة الفرعية.
3. أضف ملفات المحتوى المشار إليها في `tabs`.

## إضافة تبويب Markdown جديد

أضف عنصرًا جديدًا داخل `tabs`:

```json
{
  "code": "summary",
  "title": "ملخص سريع",
  "type": "markdown",
  "file": "summary.md"
}
```

ثم أنشئ الملف `summary.md`.

## إضافة تبويب Quiz جديد

أضف عنصرًا جديدًا داخل `tabs`:

```json
{
  "code": "quiz-2",
  "title": "اختبار إضافي",
  "type": "quiz",
  "file": "quiz-2.json"
}
```

ثم أنشئ ملف JSON جديدًا بالمصفوفة المناسبة للأسئلة.
