# الشرح النظري - سداد السندات

# تعريف المفردة

سداد السندات هو إنهاء التزام السندات عند الاستحقاق أو عند السداد قبل الاستحقاق وفق شروط العقد.

# الفكرة الأساسية

الفكرة الأساسية أن التزام السندات لا يبقى إلى الأبد، بل ينقضي عند الوفاء به، وبالتالي يعالج محاسبيًا بإزالة الالتزام وما يرتبط به من أرصدة ذات صلة.

# العناصر أو الأنواع أو المكونات

- سداد عند الاستحقاق.
- سداد قبل الاستحقاق.
- القيمة الاسمية.
- القيمة الدفترية.
- أي فرق بينهما عند السداد المبكر.

# الشروط أو القواعد أو المعالجة

- عند الاستحقاق يزال التزام السندات مقابل النقدية.
- عند السداد المبكر تقارن القيمة الدفترية بالمبلغ المدفوع.
- يرتبط ذلك بإلغاء الاعتراف بالالتزام المالي ضمن منطق `IFRS 9`.

# الخطوات العملية

1. تحديد رصيد السندات.
2. تحديد ما إذا كان السداد عند الاستحقاق أو قبله.
3. تحديد المبلغ المدفوع.
4. مقارنة القيمة الدفترية إن لزم.
5. إثبات قيد السداد.

# العلاقات مع المفردات الأخرى

- يرتبط بالسندات أصلًا.
- يرتبط بخصم أو علاوة الإصدار إذا بقي لهما أثر.
- يرتبط بالفوائد المستحقة إذا تزامن السداد مع دفع فوائد.

# النقاط التي يركز عليها الاختبار

- الفرق بين السداد عند الاستحقاق والسداد المبكر.
- إزالة الالتزام.
- أثر أي فرق عند السداد المبكر.
- الربط بـ `IFRS 9`.

# ملخص سريع جدًا

- السداد يعني إنهاء الالتزام.
- عند الاستحقاق يقفل الالتزام مقابل النقدية.
- قبل الاستحقاق قد يظهر فرق يحتاج تحليلًا.
